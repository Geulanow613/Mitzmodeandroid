# swift-native — JSON bundle for Swift Mitz Mode

Drop this folder into your Xcode project (e.g. `MitzMode/Resources/Localization/`).

## Load at startup

1. Parse `BundledTranslations/{lang}.json` → `[String: String]` keyed by English source text.
2. Parse `Mitzvot/mitzvotcloud.json` and `Mitzvot/mitzvotlistfull.json` for Mitzvah Me.
3. Parse `Checklist/checklist-items.json` for the daily checklist.
4. Parse `Rewards/mitzvah_level_thresholds.json` + translate tier keys for the counter UI.

See **`../docs/SWIFT_NATIVE_TRANSLATIONS.md`** for full integration notes.
