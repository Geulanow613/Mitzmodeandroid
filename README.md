MitzMode - Jewish Mitzvah App

MitzMode is an Android app that helps users discover and track daily mitzvot (good deeds) in Jewish practice. The app features a mitzvah generator, daily checklist for those looking to increase mitzvah observance, and more.
🎯 Features

    Mitzvah Generator: Get random mitzvot from a curated database
    Daily Mitzvah Checklist: Track daily Jewish observances for men and women
    Level System: Progress tracking with reward videos
    Background Videos: Cool animated backgrounds
    Offline Support: Works without internet connection
    Add Mitzvot: Submit new mitzvot for community use
    Multi-language Support: Translate the app into 35+ languages using Google Translate

🚀 Getting Started
Prerequisites

    Android Studio Arctic Fox or later
    Android SDK 24+ (Android 7.0)
    Java 17 or later
    Git

Installation

    Clone the repository

    git clone https://github.com/yourusername/mitzmode.git
    cd mitzmode

    Open in Android Studio
        Launch Android Studio
        Select "Open an Existing Project"
        Navigate to the cloned directory and open it

    Configure Required Settings

    a) Email Configuration (Required for "Add Mitzvah" feature)

    In app/src/main/java/com/beardytop/mitzmode/ui/components/AddMitzvahDialog.kt, replace:

    return PasswordAuthentication(
        "YOUR_EMAIL_HERE@gmail.com",
        "YOUR_APP_PASSWORD_HERE"
    )

    With your Gmail credentials:
        Use a Gmail account
        Enable 2-factor authentication
        Generate an App Password
        Replace YOUR_EMAIL_HERE@gmail.com with your email
        Replace YOUR_APP_PASSWORD_HERE with your app password

    b) Google Translate API Configuration (Optional - for translation feature)

    1. Go to Google Cloud Console: https://console.cloud.google.com/
    2. Create a new project or select an existing one
    3. Enable the Google Translate API
    4. Create credentials (API Key)
    5. Copy local.properties.template to local.properties
    6. Replace YOUR_GOOGLE_TRANSLATE_API_KEY_HERE with your actual API key

    c) Sentry Configuration (Optional - for crash reporting)

    In app/build.gradle, replace YOUR_SENTRY_DSN_HERE with your Sentry DSN:

    manifestPlaceholders = [
        SENTRY_DSN: "https://your-sentry-dsn@sentry.io/project-id"
    ]

    To get a Sentry DSN:
        Create account at sentry.io
        Create new Android project
        Copy the DSN from project settings

    App Signing (Required for release builds)

    Generate your own keystore:

    keytool -genkey -v -keystore mitzmode.jks -keyalg RSA -keysize 2048 -validity 10000 -alias mitzmode

    Update app/build.gradle with your signing configuration.

    Build and Run
        Sync the project with Gradle files
        Build and run on your device or emulator

📋 Excluded Files & Setup Required

The following files are excluded from version control for security and local configuration reasons. You'll need to create/configure them to build and run the app:

Required Files:

1. local.properties (Auto-generated by Android Studio)
   - This file is automatically created when you open the project in Android Studio
   - Contains your local Android SDK path: `sdk.dir=/path/to/your/android/sdk`
   - If it doesn't auto-generate, create it in the root directory with your SDK path

2. mitzmode.jks (Required for release builds only)
   - Keystore file for signing release APKs
   - Generate using:
     ```
     keytool -genkey -v -keystore mitzmode.jks -keyalg RSA -keysize 2048 -validity 10000 -alias mitzmode
     ```
   - Store this file securely and back it up - if lost, you cannot update your app on Google Play
   - ⚠️ NEVER commit this file to version control

Optional Files (for specific features):

3. sentry.properties (Optional - for crash reporting)
   - Copy from sentry.properties.template
   - Add your Sentry DSN and project configuration
   - Or configure directly in app/build.gradle

4. app/google-services.json (Optional - if using Firebase)
   - Download from Firebase Console
   - Place in app/ directory

5. Google Translate API Key (Optional - for translation feature)
   - Copy local.properties.template to local.properties
   - Add: `GOOGLE_TRANSLATE_API_KEY=your_api_key_here`
   - Get API key from Google Cloud Console

Quick Setup Checklist:

- [ ] Open project in Android Studio (auto-creates local.properties)

Note: The app/build/ directory is also excluded as it contains generated build files. This is normal and expected.
